# Proje Özeti

Wanderly AI Travel Planner, kullanıcıların gitmek istedikleri şehir için kişiselleştirilmiş seyahat planları oluşturmalarını sağlayan yapay zekâ destekli bir web uygulamasıdır.

Kullanıcı seyahat edeceği şehri seçtikten sonra ilgi alanlarını, seyahat süresini ve bütçesini belirler. Uygulama bu bilgiler doğrultusunda günlük gezi planı oluşturarak kullanıcıya düzenli bir rota sunmaktadır.

Projenin temel amacı, seyahat planlama sürecini kolaylaştırmak ve kullanıcıların kısa sürede düzenli bir gezi programı hazırlayabilmesini sağlamaktır.

# Kurulum

Projeyi bilgisayarınıza indirdikten sonra aşağıdaki komutları çalıştırınız.

```bash
npm install
```

Ardından geliştirme sunucusunu başlatınız.

```bash
npm run dev
```

Daha sonra tarayıcıdan aşağıdaki adresi açınız.

```
http://localhost:3000
```
